글자/문단 관련 속성

글자 관련 : *color, *font-size, *font-weight(두께), font-stretch(장평), font-lineheght(행간), font-lineheght(행간), text-transform(대-소 변환), font-variant(대문자로 변환하고 소문자 크기로), *font-style(기울기), *font-family(글꼴), font-kerning(자간) *@font-face

문단 관련 : letter-spacing(자간), word-spacing(어간), line-height(행간), text-align, *text-indent(들여쓰기/내어쓰기), *text-decoration, text-justify, text-orientation, *text-overflow, *white-space, word-break(강제줄바꾸기)



글자색

color : 웹컬러명 | 16진코드 | rgb()/rgba() | hs1()/hsla()



글자 크기

font-size : keyword(지정된 단어) | length, font

keyword : xx-small | x-small | smaller | small(한 단계 작게-상대적) | medium | large | x-large | xx-large |

length unit(단위) : px, em(rem포함), pt 등 - vw,vm은 글자에 잘 안씀

keword를 사용하여 글자 크기를 지정할 경우는 브라우저마다 모두 다르므로, length 값으로 지정하도록 하는 것이 좋다.



글자 두께

font-weight : keyword | number(단위가 안붙음)

keyword : bolder | bold | normal | lighter

number - 100 ~ 900로 100단위로 기재. 100이 가장 가늘고, 900이 가장 두꺼움.



글자 장평

font-stretch : ultra-condensed | extra-condensed | condensed | semi-condensed normal(기본값) | semi-expanded | expanded(양적확장) | extra-expanded | ultra-expanded

장평(font-stretch)이란? 한 글자의 가로크기와 세로크기의 비율이며, 기준은 세로크기에 비해 가로크기가 얼마나 큰지에 대하여 값을 부여함.



글자 스타일

font-style : normal | italic(기울기) | oblique(기울기)

기울임꼴의 유무를 나타내는 속성



글자 변형

font-variant : normal | small-caps

영문의 경우 대문자와 소문자의크기가 각각 달라서 균일하게 글자크기나 폭을 지정하기 힘드므로, 대문자를 소문자 크기로 조절하기 위한 속성



글자 변환

text-transform : none(기본값) | upppercase | lowercase | capitalize(첫자 대/소문자)

해당 단어나 문장의 영문을 대문자 또는 소문자로 변환할 때 사용



글꼴 지정

font-family : 1차폰트[, 2차폰트][, 3차폰트]

서버에 설치되어 있는 폰트의 이름을 지정하며, 만약 폰트의 이름 중간에 공백이 있거나, 한글과 같은 영문이 아닌 경우 반드시 인용부호인 ""나 ''로 감싸 표기해야함. 만약, 1차 폰트가 존재하지 않는 경우 2차 폰트가 적용되고, 2차 폰트도 적용될 수 없다면 3차 폰트가 적용됨. 다만 3차 폰트가 적용되지 못하는 경우를 대비하여 3차 폰트는 브라우저 기본 제공 폰트를 지정해야함.

브라우저에서 제공하는 기본 폰트는 해당 서버나 클라이언트에 설치가 되어 있지 않아도 자동으로 제공됨. sans-serif(고딕계열), serif(명조계열), cursive(궁서계열), monospace(가로세로 장평이 같은 폰트), fantasy(디자인계열) 등을 말함.



자간(커서 위치)

font-kerning : auto | normal | none



자간(해당 요소 안의 글자)

letter-spacing : normal | length

length : px, em, pt 등의 단위를 주로 활용

해당 요소 안의 글자와 글자 사이의 간격



어간

word-spacing : normal | length

length : px, em, pt 등의 단위를 주로 활용

단어와 단어 사이의 간격



행간

line-height : normal | number | length

값으로 숫자만 기재할 경우 현재 문단의 글자크기의 배수가 됨. 숫자 뒤에 단위가 들어가는 경우는 해당하는 단위의 크기가 적용됨. (글자10p일 때 행간 값 5입력하면 행간은 50px 나옴)



문단 정렬

text-align : left | right | center | justify(배분정렬, 나눔정렬)

문단 가로방향에 대한 정렬



들여쓰기/내어쓰기

text-indent : length

length 단위 : px, %, pt 등

들여쓰기 : 다른 줄에 비해 첫 번째 줄이 더 들어가게 쓰는 것
text-indent: 20px;

내어쓰기 : 다른 줄에 비해 첫 번째 줄이 더 나오게 쓰는 것
text-indent: -20px;



글자 장식

text-decoration : none(기본값) | underline(a태그의 기본값) | overline | line-through(취소선,가운데선)

글자의 위, 중간, 아래에 줄을 표시하도록 하는 속성

none을 제외한 다른 값은 중복해서 사용할 수 있음.
text-decoration:overline underline
text-decoration:underline double #369, overline wave #c00;

세부속성

text-decoration-color : 줄의 색상

text-decoration-line : 줄의 위치(underline / overline / line-through)

text-decoration-style : 줄의 종류(solid(실선) / wave(물결) / double / dotted / dashed)



배분정렬

text-justify : auto(영문만) | none(한글) | inter-character(글자가 벌어짐) | inter-word(단어간 간격만 벌어짐)

단어 또는 글자 단위로 한 줄의 데이터를 동일한 분량으로 배분정렬할 수 있는 속성



글자 진행 방향 (읽는 방향, 쓰는 방향 모두 해당)

text-orientation : mixed | upright | sideways(어느한쪽 쏠리게) | sideways-right | use-glyph-orientation(사용자가 쓰는 폰트에 맞춰서)



글자 흘러넘침

text-overflow : clip(기호) | ellipsis | 특정문자

text-overflow는 overflow:hidden이며, white-space:nowrap인 경우에 유효함.



내용 보호 or 줄의 끝 처리

white-space : normal | nowrap(무조건 여러 줄이 한 줄이 됨) | pre(입력한 거 그대로 출력) | pre-line(줄속성 지켜짐, 줄바꿈) | pre-wrap(줄바꿈도 되고 입력한 거 그대로 출력)

하나의 단어가 한 줄에 입력되어야만 하기 때문에, 한 줄에 단어가 배치가 되던지, 영문 배치문법을 무시하고 단어 중간을 자를지, 하나의 문장을 한 줄로만 표시할 지 그 문장의 줄의 끝을 어떻게 처리할 지 정하는 속성



단어 처리

word-break(강제줄바꾸기) : normal | break-all | keep-all | break-word |

보통 영문 처리할 때 사용, 한 줄 또는한 문장 안에서의 단어의 끝처리 부분을 어떻게 할지 정하는 속성
